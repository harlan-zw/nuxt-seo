# Awesome Blog

Welcome to a tiny blog built with Nuxt Content and `@nuxtjs/seo`.

Every page automatically gets:

- A `<title>`{lang="html"} and `<meta name="description">`{lang="html"} from frontmatter
- An OG image generated by `nuxt-og-image`
- An entry in `/sitemap.xml`
- Schema.org metadata via `nuxt-schema-org`
