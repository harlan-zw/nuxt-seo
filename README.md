## Installation

Install `@nuxtjs/seo` dependency to your project:

```sh
npx nuxi@latest module add seo
```

Once installed check the [Default Module Features](https://nuxtseo.com/docs/nuxt-seo/guides/using-the-modules) guide
for a quick start.
